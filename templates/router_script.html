路由器 NATMap 通知脚本

以下脚本已自动填充服务器地址为 {{ server }}，请根据你的节点数量分别创建并赋予执行权限。

{% for node in nodes %}
节点 {{ node.id }}：{{ node.name }} 
#!/bin/sh
# 节点 {{ node.id }} 通知脚本（由 NATMap 自动调用）
# 参数说明：$1 = 外部IP, $2 = 外部端口
curl -s "http://{{ server }}/setip?ip=$1"
curl -s "http://{{ server }}/set{{ node.id }}?p=$2"

保存为 /usr/bin/notify{{ node.id }}.sh 并添加执行权限：

chmod +x /usr/bin/notify{{ node.id }}.sh
{% endfor %}
（可选）单独上报公网 IP 的定时任务脚本 
#!/bin/sh
# 获取公网 IPv4 地址并上报（若 NATMap 已上报 IP 可省略）
PUBLIC_IP=$(curl -s -4 ifconfig.me)
[ -n "$PUBLIC_IP" ] && curl -s "http://{{ server }}/setip?ip=$PUBLIC_IP"

保存为 /usr/bin/report_ip.sh，添加执行权限，并添加 crontab 任务（例如每5分钟执行一次）：

chmod +x /usr/bin/report_ip.sh
crontab -e
# 添加以下行
*/5 * * * * /usr/bin/report_ip.sh >/dev/null 2>&1

    • {% endfor %}

启用实例后，当 NATMap 成功映射时，脚本会自动上报公网 IP 和端口到服务器。
